# 💰 AI Financial Optimizer

An AI-powered financial decision system that analyzes user income and spending behavior, applies government tax policies, and generates personalized financial insights and reports.

---

## 🚀 Features

- 📊 Interactive dashboard for financial analysis  
- 🧠 AI-driven insights on savings and spending  
- 🏛️ Government policy engine for tax optimization  
- 📈 Investment allocation based on risk profile  
- 📄 Professional PDF report generation  
- 📥 CSV export functionality  

---

## 🧠 How It Works

1. User inputs income and expenses  
2. System calculates financial metrics  
3. Policy engine applies tax rules and investment logic  
4. Dashboard visualizes insights  
5. Report module generates a structured financial report  

---

## 📊 Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib  
- ReportLab  

---

## 🏛️ Policy Engine

The system includes a modular policy engine that simulates:
- New vs Old tax regime comparison  
- Risk profiling based on savings behavior  
- Investment allocation strategy
